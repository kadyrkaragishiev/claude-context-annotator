## [0.1.0] - 2026-03-16

### Added
- Initial release
- Annotate code regions with labels via `Cmd+Shift+A` / `Ctrl+Shift+A`
- Sidebar panel to accumulate and manage context blocks
- Selective sending — checkboxes to include/exclude blocks
- Send to Claude Code directly or copy to clipboard as fallback
- Click file path in sidebar to navigate to code region
- Deduplication — re-annotating the same region updates existing block
